# 🍽️ Bella Vista Restaurant Website

Bella Vista is a modern and responsive restaurant website built using HTML and CSS. This project showcases a clean and elegant user interface for a restaurant, including sections like menu, about, services, and contact.

---

## 📌 Features

- Beautiful and modern UI design  
- Fully responsive layout  
- Navigation bar with smooth scrolling  
- Home, About, Menu, Services, and Contact sections  
- Stylish food menu display  
- Contact form UI  
- Clean and organized code  

---

## 🛠️ Technologies Used

- HTML5 – Structure of the website  
- CSS3 – Styling and layout design  

---

## 📂 Project Structure

Bella-Vista-Website/
│
